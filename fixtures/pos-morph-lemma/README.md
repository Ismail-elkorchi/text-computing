# POS, morphology, and lemma fixtures

This fixture set records readiness artifacts for issue `#10`.

- `slices.json` freezes the input slices that feature work must cover.
- `tool-versions.json` freezes comparator versions and the target annotation inventory.
- future `expected/` artifacts will use the schema already defined in
  `schemas/pos-morph-lemma-expected-v1.schema.json`.

This readiness set does not yet claim implemented POS, lemma, or morphology behavior.

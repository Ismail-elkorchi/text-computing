# POS, morphology, and lemma fixtures

This fixture set records the public readiness and feature-verification artifacts for issue `#10`.

- `slices.json` freezes the input slices that feature work must cover.
- `tool-versions.json` freezes comparator versions and the target annotation inventory.
- `expected/` records the current deterministic goldens generated by `@ismail-elkorchi/textrules`.
- `manifests/` and `resources/` contain the pack-backed lexicon fixtures consumed by the analyzer.
- recorded `expected/` artifacts use the schema defined in
  `schemas/pos-morph-lemma-expected-v1.schema.json`.
